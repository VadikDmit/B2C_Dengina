# Anna Dengina: Premium AI Financial Planning Ecosystem

Интерактивная экосистема для управления личными финансами, построенная вокруг человечного опыта и глубокой аналитики под руководством Искусственного Интеллекта.

---

## 🌟 Концепция проекта
Это не просто «финансовый калькулятор». Это **цифровой ассистент**, который снимает барьер страха перед сложными цифрами. Мы объединяем экспертизу Анны Деньгиной с мощью ИИ, чтобы создать понятный, наглядный и «живой» финансовый план.

## 🧱 Основные модули

### 1. ⏪ Прошлое (Статистика и Контекст)
Раздел, погружающий пользователя в текущую рыночную ситуацию.
- Анализ инфляции, ключевых ставок, динамики ОФЗ и металлов.
- Позволяет понять, почему ИИ принимает те или иные решения в расчетах.
- ИИ-сводки с комментариями от Анны по текущим трендам.

### 2. 📍 Настоящее (Портфель и Защита)
Наглядная визуализация текущего финансового состояния.
- Распределение активов (круговые и столбчатые диаграммы).
- Блок «Защита»: оценка финансового резерва и страхового покрытия.
- Прямая связь между текущими накоплениями и будущими целями.

### 3. ⏩ Будущее (Персональный финансовый план)
Ядро системы, где происходит моделирование жизни.
- Расчет достижения целей (Пенсия, Недвижимость, Образование).
- Динамический пересчет планов при изменении вводных данных.
- Визуализация вероятности успеха и рекомендации ИИ по оптимизации.

### 4. 💬 Интерактивный онбординг (Виктория)
Дружелюбный вход в систему для новых клиентов.
- Чат-анкета с ИИ-помощницей Викторией.
- Сбор базовых данных (имя, возраст, пол) в формате живого диалога.
- Ползунки и кнопки вместо скучных форм ввода.

## 🛠 Технологический стек
- **Frontend:** React 19 + TypeScript.
- **Styling:** Vanilla CSS с упором на **Premium UI** (стекляные эффекты, градиенты, глубокие тени).
- **Animations:** Framer Motion для плавных переходов и «живого» интерфейса.
- **Charts:** Recharts для профессиональной визуализации данных.
- **AI Integration:** Собственные API-методы для стриминга ответов и генерации контекстных советов.

## 🎨 Дизайн-философия (Premium Aesthetics)
Проект следует принципам современного премиального дизайна:
- **Glassmorphism:** Использование эффектов размытия заднего фона.
- **Micro-animations:** Тонкие реакции интерфейса на действия пользователя.
- **Curated Palette:** Использование фирменных цветов (Primary: #FFC750) и гармоничных градиентов.

## 🚀 Запуск проекта
1. Установите зависимости: `npm install`
2. Запустите сервер для разработки: `npm run dev`
3. Сборка проекта: `npm run build`
